# Pantau Stunting (Indonesia Stunting Tracker)

An interactive analytical dashboard designed to monitor, visualize, and analyze the prevalence of child stunting across 34 provinces in Indonesia (2023 Data). 

Built as a final project for the **Official Statistics** course, this platform transforms complex demographic, socio-economic, and infrastructure data into actionable insights. It features multivariable clustering, spatial autocorrelation (Moran's I), and OLS predictive regression models to uncover the structural root causes of stunting—such as the massive impact of poverty and the infrastructure paradox.

## 🚀 Key Features
- **Interactive Regional Mapping:** Deep dive into specific islands and provinces.
- **Socio-Economic Determinants:** Scatter charts analyzing poverty, food inadequacy (PoU), and sanitation.
- **Advanced Data Science POV:** K-Means clustering, predictive modeling, and spatial analysis natively interpreted for policy making.
- **Modern UI/UX:** Built with React, Vite, Recharts, and a clean Glassmorphism aesthetic.

## 🛠️ Technology Stack
- **Frontend Framework:** React + Vite
- **Styling:** Vanilla CSS (Glassmorphism design system)
- **Data Visualization:** Recharts
- **Icons:** Lucide React
